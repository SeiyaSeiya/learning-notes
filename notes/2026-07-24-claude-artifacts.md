# Claudeのアーティファクト（Artifacts）とは何か

- 学んだきっかけ: Coworkとの会話中に「Claudeのアーティファクトは実際にWeb上に公開されているのか」という疑問から、アーティファクト機能の仕組みと利用例を調べた。

## 1. アーティファクトとは何か

- Claudeとの会話の中で、コードや文書、図表などのまとまったコンテンツを、チャット本文とは別のプレビューパネルに表示する機能。
- 単発の質問への回答ではなく、ある程度まとまった「成果物」を作るときに使われる。長めのコード、Web上で動くツール、レポートなど、後でコピーしたり操作したりすることが前提のコンテンツが対象。
- 対応するコンテンツ形式: Markdown文書（`.md`）、HTML（CSS/JS込みの単一ファイル）、Reactコンポーネント（`.jsx`）、SVG画像、Mermaid図、PDF。

## 2. Web上に公開されるものではない（重要な誤解ポイント）

- アーティファクトは会話内でのみ表示されるプレビューであり、外部から誰でもアクセスできる公開URL（独自ドメインでホスティングされたWebサイトのようなもの）は発行されない。
- 表示はサンドボックス化された環境（iframe的なもの）でレンダリングされているだけ。
- 共有できる範囲は、その会話にアクセスできる人、または明示的な共有リンクを知っている人に限られる。検索エンジン経由で誰でも見つけられるような公開サイトにはならない。
- 社内資料や個人ブログのように、独自ドメインで恒常的にホスティングされる状態とは異なる。

## 3. 利用例（詳細）

アーティファクトが向いている代表的なユースケースは以下の通り。

| 用途 | 使う形式 | 具体例 |
|---|---|---|
| Webサイト・ツールの試作 | HTML | ランディングページの試作、フォーム、計算ツール（ローン計算機、単位変換ツールなど） |
| データ可視化・ダッシュボード | React（recharts, d3, Chart.jsなど利用可） | 売上データのグラフ化、KPIダッシュボード、CSVを読み込んで可視化するツール |
| インタラクティブなアプリ・ゲーム | React / HTML | 簡単なゲーム、シミュレーター、ミニアプリ |
| 長文コード・コンポーネント | 各種プログラミング言語 / React | 再利用可能なUIコンポーネント、まとまった量のスクリプト（10行を超えるようなもの） |
| レポート・記事・ガイド | Markdown | ブログ記事、業務マニュアル、比較レポートなど「後でコピーして外部で使う」文書 |
| 図解・フローチャート | SVG / Mermaid | 業務フローの図示、アーキテクチャ図、組織図 |
| 資料の一部としての図表 | SVG | イラスト、アイコン、概念図 |

- 特徴的なのは、HTML/Reactアーティファクトは**実際に動く**点。ボタンを押す、入力フォームに値を入れる、グラフをホバーする、といった操作がプレビュー内でそのまま可能。
- Reactアーティファクトでは`lucide-react`（アイコン）、`recharts`（グラフ）、`d3`、`three.js`（3D）、`Tone`（音声）、`mathjs`、`lodash`、`Papaparse`（CSV処理）、`SheetJS`（Excel処理）、`shadcn/ui`などのライブラリが利用可能で、単なる静的コードだけでなく、ある程度本格的なミニアプリケーションが作れる。
- 編集を依頼すると、Claudeは同じアーティファクトを更新し、バージョン履歴として過去の版も遡って確認できる。
- ブラウザのlocalStorage/sessionStorageは使用不可（Claude.ai環境で動作しないため）。データの保持が必要な場合はReactのstateやメモリ内変数で対応する設計になっている。

## 4. アーティファクトを使わないケース

- 短い会話的な回答、その場で消費されて終わる説明には使われない。
- 既存の学習知識で答えられる事実質問（例:「◯◯の意味は？」）にはアーティファクトを起こさず、通常の文章で回答する。
- コンテンツが「後でコピーして使う」「見返す」性質のものでなければ、わざわざ別パネルに分離する必要はない。

## 5. まとめ

- アーティファクト＝会話に紐づく「プレビュー可能な成果物」であり、独立したWebサイトとして公開される仕組みではない。
- 向いているのは、コード・ツール・図表・長文文書など、単独で完結して見返す価値があるコンテンツ。
- HTML/Reactでは実際に操作できる点が、単なるコードスニペット表示との大きな違い。
